# Rock Paper Scissors Game

A beginner-friendly Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Live Demo
https://mohammad-aahil.github.io/rock-paper-scissors/

## Features
- User vs Computer gameplay
- Auto Play mode
- Score tracking using localStorage
- Reset functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes
- Improved JavaScript logic building
- Better understanding of DOM manipulation
- Hands-on experience with browser localStorage
