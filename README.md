# Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Live Demo
https://mohammad-aahil.github.io/rock-paper-scissors/

## Features
- User vs Computer gameplay
- Dynamic result display with images
- Auto Play mode
- Score tracking using localStorage
- Reset functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes
- Strong understanding of relative paths
- JavaScript logic building and conditionals
- DOM manipulation and dynamic content rendering
- Hands-on experience with localStorage
