# Rock Paper Scissors Game

A simple Rock Paper Scissors game developed using HTML, CSS, and JavaScript.
This project was built to practice JavaScript logic, DOM manipulation, and
event handling.

## Features
- User vs Computer gameplay
- Random computer choice generation
- Dynamic result display
- Clean and responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes
- Improved JavaScript logic building
- Understanding of conditionals and functions
- Hands-on experience with DOM manipulation

## Future Improvements
- Score tracking
- Sound effects
- Improved UI animations
