# Rock Paper Scissors Game

A beginner-friendly Rock Paper Scissors game built using HTML, CSS, and JavaScript.
This project focuses on JavaScript logic building, DOM manipulation, and event handling.

## Live Demo
https://mohammad-aahil.github.io/rock-paper-scissors/


## Features
- User vs Computer gameplay
- Random computer choice generation
- Dynamic result display
- Clean and responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes
- Improved JavaScript logic building
- Understanding of conditionals and functions
- Hands-on experience with DOM manipulation

## Future Improvements
- Score tracking
- Sound effects
- Improved UI animations
